# Progress-tracking-in-project-management
## [实验任务一]：Project 2010中进度跟踪
1.打开上次上机时提交的Project 源文件
2.在任务工作表中加入列“比较基准开始时间”和“比较基准结束时间”
3.自己填写每个活动的 “比较基准开始时间”和“比较基准结束时间”
要截图的输出：
1. 跟踪甘特图
2. 表格：视图->表格->差异 
注意：1. 比较基准为项目的计划；
     2. 阅读Project 2007 简体中文教程第6章
